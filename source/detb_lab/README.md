# DETB Lab

External Isaac Lab extension package for DETB-owned tasks, robot configs, and simulator-native helpers.

Current owned modules:

- `detb_lab.assets.robots.anymal_c`: DETB baseline ANYmal-C articulation and actuator config.
- `detb_lab.tasks.manager_based.locomotion.velocity.config.anymal_c`: DETB task and PPO configs for the migration anchor.
- `detb_lab.tasks.manager_based.locomotion.velocity.config.anymal_c_stability`: DETB stability-focused locomotion task family.
- `detb_lab.tasks.manager_based.locomotion.velocity.config.anymal_c_simple_actuator`: DETB locomotion family for the simple DC actuator profile.
